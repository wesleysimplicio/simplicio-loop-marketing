The problem

  • Most AI-marketing tooling asks operators to trust a black box.
  • Prompts are vague, provider coupling is hidden, evidence is optional.
  • Public claims routinely outrun what the underlying repo can actually prove.

The promise

  • Routing is documented — vendor swaps are configuration, not rewrites.
  • Artifacts are local and inspectable — no hidden cloud state.
  • Campaign planning is a bounded set of documents, not tribal knowledge.
  • Promotion is gated behind budget guardrails and human approval.
  • Lessons persist across iterations via a durable, deduped journal.

What is still external

This page is a repo-local, self-contained static asset — it renders correctly opened directly from disk or served by any static host. Deploying it at a real domain (for example simpleti.com.br/simplicio/on-metal), wiring site navigation, and collecting production analytics remain external follow-up: they depend on a real hosting target this repository does not own.
