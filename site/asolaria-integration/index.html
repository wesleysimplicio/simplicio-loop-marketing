Simplicio + Asolaria — Agente Evolutivo no Metal

Section prepared for the Simplicio marketing site (P2, issue #78): how the Simplicio ecosystem's own maps, reductions, and provider-agnostic runtime read against the Asolaria narrative ("map of maps", "reductions in everything", "ASI on metal, fabric and matrix").

Integration timeline

  • Repo-local maps existed first
    VISION / DOMAIN / DESIGN / PROVIDERS / PLAYBOOKS specs, pre-dating this campaign

    The engine already tracked product, architecture, operator, and strategy maps separately — the raw material Asolaria's "map of maps" framing needed a canonical index for.

  • SIMPLICIO-MAP-OF-MAPS.md — canonical bridge
    SIMPLICIO-MAP-OF-MAPS.md

    One index that names every local map, what it is for, and how each maps onto the Asolaria narrative — without claiming capability the repo doesn't have.

  • REDUCTIONS.md — the reduction catalog
    REDUCTIONS.md — 6 reductions, each with a repo proof trail

    "How do we get reductions in everything" becomes six concrete, file-and-test-backed reductions: provider routing, campaign bounding, evidence-aware content, receipt-based completion, guarded promotion, and looped iteration.

  • Reproducible demo + benchmark scripts
    scripts/demo-asolaria-loop.mjs, scripts/reductions-benchmark.mjs

    "On metal" becomes literal: these are plain Node scripts anyone can run locally, with no cloud dependency, that measure real repo files instead of asserting numbers.

  • Simplicio on Metal landing page
    site/simplicio-on-metal/index.html

    Self-contained static page, ready to deploy at a real domain once one exists — the last repo-local step before public publishing.

Boundary: narrative inspiration vs capability claim

This section borrows Asolaria's framing language ("map of maps", "reductions", "on metal") as narrative structure. It does not claim ASI, autonomous self-rewriting, or any capability beyond what REDUCTIONS.md and the linked tests can demonstrate today. See HYPOTHESIS.md for the falsifiable claims this narrative rests on.
